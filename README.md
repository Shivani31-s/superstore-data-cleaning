# Superstore Data Cleaning Project

## Dataset Used
Superstore sales dataset containing order, customer, product, and sales details.

## Data Cleaning Steps Performed
- Fixed date formats for order_date and ship_date
- Verified numeric columns (sales, profit, discount, quantity)
- Checked for missing values (no missing values found)
- Checked and removed duplicate order IDs
- Added Data_Quality_Notes column to document cleaning steps

## Files in this Repository
- Raw_Data.xlsx – original dataset
- Cleaned_dataset.xlsx – cleaned data with analysis sheets
- Cleaned_dataset.csv – cleaned data (single sheet)
